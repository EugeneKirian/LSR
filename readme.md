# Learn Software Rasterization

![LSR](./Images/screenshot.png)

## Description
A simgle-threaded 3d software rasterizer developed for learning purposes.

## Features
1. Rasterizes dots, lines, and triangles.
2. Supports clipping and culling.
3. Supports fog effect.
4. Suports various fill methods for triangle rasterization.
